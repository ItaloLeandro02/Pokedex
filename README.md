<h3>Instalação</h3>
<p>

```diff
git clone https://github.com/ItaloLeandro02/Pokedex.git
```
</p>

<h3>Dependências</h3>
<p>

```diff
npm i ou yarn
```
</p>

<h3>Execução</h3>
<p>

```diff
npm run android && npm run start ou  yarn android && yarn start
```
</p>


<h3>Principais Tecnologias</h3>

* [Axios](https://github.com/axios/axios)
